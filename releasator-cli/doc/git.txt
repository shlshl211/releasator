How to map Git urls to VcsId

GIT url examples:
scm:git:git://git.cedarsoft.com/com.cedarsoft.open
scm:git:ssh://git.cedarsoft.com/home/git/com.cedarsoft.open
scm:git:git://github.com/talios/clojure-maven-plugin.git
scm:git:git@github.com:pkozelka/git-demo.git/submodule


CVS url examples
scm:cvs:pserver:anonymous@patterntesting.cvs.sourceforge.net:/cvsroot/patterntesting:PatternTesting08/patterntesting-samples


Commandline to make the release:

releasator --author pkozelka@gmail.com full vcs:bitbucket.org/pkozelka/buildbox:master/releasator-cli 1.0.2-alpha-1
releasator --author pkozelka@gmail.com full vcs:github.com/buildbox/contentcheck-maven-plugin:master 1.0.3

releasator --author petr.kozelka@hp.com full vcs:czvm133/petr.kozelka/test:master 1.0.0


[bitbucket.org]
vcs.type=git
connection.mask=git@bitbucket.org:[REPOURI].git
web.mask=https://bitbucket.org/[REPOURI]
web.software=bitbucketweb:0

[github.com]
vcs.type=git
connection.mask=git://github.com/[REPOURI].git
web.mask=https://github.com/[REPOURI]
web.software=githubweb:0
filerelease.list.url=https://github.com/[REPOURI]/downloads
filerelease.url=https://github.com/[REPOURI]/downloads/[FILE]

[sourceforge.net]
vcs.type=svn
connection.mask=https://[REPOURI].svn.sourceforge.net/svnroot/[REPOURI][PATH]
web.mask=
web.software=viewvc:1.0.7
filerelease.url=
